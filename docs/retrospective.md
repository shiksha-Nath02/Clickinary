# Retrospective

## Lesson 1
Working with external APIs requires careful understanding of the API response structure. Reading API documentation and inspecting responses is essential for building reliable integrations.

## Lesson 2
Dynamic UI rendering is much easier when the interface is built using reusable components such as cards or structured sections.

## Lesson 3
Error handling is an important part of frontend development. Applications should always handle situations where data is missing or API requests fail.

## Lesson 4
Asynchronous programming is fundamental when building applications that interact with APIs. Understanding promises and async/await greatly simplifies data fetching logic.

## Lesson 5
User experience improves significantly when the interface is simple, organized, and responsive. Structuring UI elements clearly makes applications easier to use.
